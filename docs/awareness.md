# Version 0.1.0

### Awareness by push

| System Event                              | Awareness element                                                                                                                                                       |
| ----------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| When the user connects                    | Abstract sound comparable with a bip (Earcon) + text-to-speech number of participants + text-to-speech word "participants" + Text-to-speech name of each connected user |
| When another user connects or disconnects | text-to-speech of the user name who enters or leaves + text-to-speech of the word connect/disconnect                                                                    |
| Send Message                              | Abstract sound similar a soap ball bursting (Earcon) transformed by one effect + text-to-speach name of message owner                                                   |
| Receive Message                           | Abstract sound similar a soap ball bursting (Earcon) transformed by one effect                                                                                          |
| Typing                                    | Concrete sound of keyboard typing (Auditory Icon) transformed by one effect                                                                                             |
| Start editing                             | Abstract sound (Earcon) transformed by one effect                                                                                                                       |
| End editing                               | Abstract sound (Earcon) transformed by one effect                                                                                                                       |

### Sound awareness by pull
| Shortcut            | Awareness element                                                     |
| ------------------- | --------------------------------------------------------------------- |
| shift + arrow up    | Describes the participants status through text-to-speech              |
| shift + arrow down  | Describes the rubric status through text-to-speech                    |
| shift + arrow right | Reads the last five messages of the Sound Chat through text-to-speech |
| shift + arrow left  | Moves the cursor (focus) to the Sound Chat input message component    |

### Available sound effects (0.1.0 and 0.2.0)
* Clean (no effect)
* Delay
* Reverb
* Wahwah
